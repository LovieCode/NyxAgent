# NyxAgent 页面级 UI 设计稿 v5

日期：2026-07-06

## 本轮修正

- 每个界面单独出图，不再把多个页面挤在一张设计图里。
- 聊天类子级页面不放底部导航，避免和页面层级冲突。
- 私聊、群聊页面补齐输入区与工具栏展开态，方便直接落地交互。
- 风格延续当前首页方向：淡雅粉色系、可爱但不过度低龄化，减少“AI 工具后台感”，保留普通聊天 App 的亲和感。
- 工具栏按现有功能做用户侧轻量命名，实际文案以实现为准。
