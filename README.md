# NyxAI 眠汐AI

基于 uni-app x 的 AI 智能体助手，支持 ReAct 代理循环、多模态模型、文件管理、记忆系统与插件扩展。

## 功能

- **AI 对话** — 流式流式输出，支持多轮对话与上下文管理
- **多模型提供商** — OpenAI、OpenRouter 等兼容接口，可自由切换模型
- **视觉识别** — 发送图片，支持多模态模型分析
- **ReAct 代理** — 内置代理循环，可自主调用工具完成任务
- **智能体技能** — 支持技能/插件系统，可扩展 Agent 能力
- **记忆系统** — 跨会话记忆存储与检索
- **文件管理** — 内置文件浏览器，管理设备文件
- **待办与笔记** — 简单的任务与笔记管理
- **自定义配置** — 模型参数、提示词、上下文长度等均可调

## 技术栈

- **框架**: uni-app x (Vue 3 + UTS)
- **语言**: UTS（跨平台强类型语言）
- **平台**: Android / iOS / Web

## 开发

```bash
# 安装依赖
npm install

# HBuilderX 中运行/打包
```

项目需使用 [HBuilderX](https://www.dcloud.io/hbuilderx.html) 打开并运行。
